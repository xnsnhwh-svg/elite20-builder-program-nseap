# Coding Coach Agent

## Purpose

The Coding Coach Agent helps students complete Challenges.

In plain language: it is the AI coach that helps students understand, build, debug, and document their work.

## Responsibilities

- Explain Challenge requirements.
- Help students create an implementation plan.
- Help set up project structure.
- Help debug errors.
- Review README and documentation.
- Check whether required submission files are present.
- Suggest improvements before final submission.

## Inputs

- Challenge description
- student repo
- error logs
- README
- AI logs
- Rubric

## Outputs

- implementation plan
- debugging suggestions
- README improvement suggestions
- pre-submission checklist
- improvement report

## MVP Behavior

For the MVP, this agent should act before Evaluation Agent. Its job is to help students improve before formal grading.
