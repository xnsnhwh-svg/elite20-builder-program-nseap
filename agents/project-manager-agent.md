# Project Manager Agent

## Purpose

The Project Manager Agent helps the class run the Builder workflow.

In plain language: it manages progress, tasks, deadlines, and review status.

## Responsibilities

- Create or summarize Challenge tasks.
- Track who has submitted.
- Track which submissions need review.
- Identify blocked students or teams.
- Generate weekly progress summaries.
- Remind students about deadlines.
- Help instructors see the class status quickly.

## Inputs

- Challenge list
- GitHub issues
- Pull requests
- submission records
- review records
- deadline calendar

## Outputs

- progress dashboard
- weekly report
- missing submission list
- blocked task list
- review queue

## MVP Behavior

For the MVP, this agent can be implemented as a prompt-driven workflow before becoming a real automated service.
