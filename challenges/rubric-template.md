# Rubric Template

Total: 100 points

| Dimension | Points | What Good Looks Like |
|---|---:|---|
| Task completion | 30 | The core task is completed and the result is usable. |
| AI collaboration process | 20 | The student shows prompts, tools, models, iterations, failures, and improvements. |
| GitHub / file structure | 15 | The repo or submission is organized, named clearly, and easy to inspect. |
| Documentation quality | 15 | README and supporting docs explain how to understand, run, or reuse the work. |
| Reflection / AAR | 10 | The student reflects on what worked, what failed, and what to improve next. |
| Reusable value | 10 | The work can help future students, teachers, or agents. |

## Rating Levels

| Level | Score Range | Meaning |
|---|---:|---|
| Excellent | 90-100 | Strong result, reusable, well documented, and thoughtfully reflected. |
| Good | 75-89 | Complete and clear, with minor gaps. |
| Pass | 60-74 | Meets basic requirements but needs improvement. |
| Revise | 0-59 | Missing key requirements or not yet usable. |
