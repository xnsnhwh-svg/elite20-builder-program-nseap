# Contribution Guide

## Basic Rules

- Use clear file names.
- Include your name and Challenge ID in submissions.
- Keep README files understandable.
- Record AI collaboration honestly.
- Prefer useful sharing over polished but unverifiable claims.

## Pull Request Checklist

- The task is linked to a Challenge.
- Required files are included.
- The README explains the work.
- The AI log is included.
- The reflection is included.
- The submission is ready for peer review.
