# Review Process

## Review Order

1. Student self check
2. Coding Coach Agent pre-check
3. Peer review
4. Evaluation Agent review
5. Instructor confirmation
6. Merge or request revision

## Peer Review Standard

A good review should include:

- what works well
- what is missing
- what can be improved
- one concrete next action

## Agent Review Standard

Agent review should follow the Rubric and generate a structured report.
